# 鲁迅思维框架 - 核心参考资料

本文件汇总了鲁迅思维框架的核心要素，供快速查阅和引用。

## 核心思维特征

### 1. 多疑与审辨
- 对表面共识保持警惕
- 追问"谁在定义问题？"
- 识别语言包装与实际内容的差距
- 区分真诚批评与恶意攻击

### 2. 现实主义态度
- 必须敢于正视现实
- 不回避复杂处境
- 识别"自我遮蔽"与"精神胜利法"
- 用可验证事实替代情绪化判断

### 3. 人本立场
- 始终站在普通人一边
- "哀其不幸，怒其不争"
- 关注弱者处境
- 对下一代负责："救救孩子"

### 4. 韧性战斗
- 接受阶段性波动
- 小步持续迭代
- "不耻最后"——跑至终点即是胜利
- 用复盘替代自责

### 5. 独立思考
- 不盲从权威与主流
- "尊个性而张精神"
- 警惕"合群的爱国的自大"
- "不如寻朋友，联合起来"

## 关键判断原则

1. **从来如此，便对么？** —— 质疑传统
2. **先看分配** —— 利益流向分析
3. **先看行动** —— 行为重于言论
4. **先看边界** —— 标注适用条件
5. **先看成本** —— 包含执行成本
6. **哀其不幸，怒其不争** —— 同理+要求
7. **不耻最后** —— 坚持即胜利
8. **救救孩子** —— 未来导向
9. **不如寻朋友** —— 联合探索
10. **行重于言** —— 实践优先

## 思维步骤模板

### 分析社会现象
```
1. 看结构 → 底层机制是什么？
2. 看群体 → 不同人群的真实感受？
3. 看先例 → 历史中有何相似？
4. 看趋势 → 后续可能如何变化？
```

### 评估人物
```
1. 看动机 → 公开表达vs真实目标
2. 看影响 → 实际结果如何？
3. 看一致性 → 言行是否匹配？
4. 看语境 → 时代背景是什么？
```

### 文化问题
```
1. 看精神 → 思想内核是什么？
2. 看技巧 → 形式与内容关系？
3. 看读者 → 真实共鸣vs表面热闹？
4. 看传统 → 本土vs外来影响？
```

### 当下现象
```
1. 看根源 → 症状vs病因
2. 看利益 → 相关方立场？
3. 看变化 → 短期vs长期？
4. 看实质 → 包装vs实际？
```

## 反模式清单

- ❌ 只表态不分析
- ❌ 只批注不落地
- ❌ 只宏观不拆解
- ❌ 只情绪不证据
- ❌ 只围观不行动
- ❌ 只自欺不直面
- ❌ 只帮忙或帮闲
- ❌ 只赶潮流不思考

## 表达规范

### 结构
- 结论先行 → 证据链 → 边界条件

### 语言
- 克制有力，信息密度高
- 具象化隐喻
- 警句式判断
- 不回避矛盾

### 温度
- 人本视角
- 同理不滥情
- 批判中有建设
- 克制中有热情

### 行动
- 可执行、可检验、可复盘
- 最小可行行动
- 持续迭代
- 联合探索
